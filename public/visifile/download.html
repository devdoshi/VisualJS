Choose your platform
Runs on Windows, Mac, Linux, Kubernetes, Docker, Snap, OpenShift, NodeJS
NodeJS
git clone https://github.com/yazz/yazz.git
cd visualjavascript
npm install
npm start
Open in a browser: http://0.0.0.0:80
Docker
docker rm vjs
docker run -p 80:80 -v ~/:/home/node/localpc --name vjs --env virtualprocessors=16 yazzcom/visualjavascript:2021
Open in a browser: http://0.0.0.0:80
OpenShift
oc new-app https://raw.githubusercontent.com/yazz/visualjavascript/master/deploy_openshift_template_vjs.yml
Windows .exe Binary
vjs.exe
Linux
wget https://yazz.com/visifile/bin/vjs
chmod +x ./vjs
./vjs
Mac
curl https://yazz.com/visifile/bin/mac/vjs -o ./vjs
chmod +x ./vjs
./vjs
